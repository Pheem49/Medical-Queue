# Frontend Documentation



## สมาชิกในทีมและการแบ่งงาน (Team Members & Route Assignments)

| ลำดับ | ชื่อ-นามสกุล | บทบาทและหน้าที่รับผิดชอบ (Role & Responsibilities) | ไฟล์และ API ที่เกี่ยวข้อง |
| :---: | :--- | :--- | :--- |
| **1** | **นางสาวกฤติมา ทองมวล (แป้ง)** | **[ Login, Register ]**<br><br>• ระบบ Authentication<br>• หน้าเข้าสู่ระบบ (Login Page)<br>• หน้าลงทะเบียนผู้ป่วยใหม่ (Registration Page)<br>• หน้าเข้าสู่ระบบของเจ้าหน้าที่ (Admin Login) | **Frontend:** `index.html`, `register.html`<br><br>**Backend:** `backend/routes/auth.py`<br><br>**API:**<br>• `POST /api/login`<br>• `POST /api/register`<br>• `POST /api/admin/login` |
| **2** | **นายจิรัฐิติกาล ลาเลิศ (บอล)** | **[ Booking, Confirm ]**<br><br>• เลือกแผนก/แพทย์<br>• หน้าหน้าตรวจสอบและยืนยันข้อมูลการจอง<br>• หน้าแสดงตั๋ว/QR Code เมื่อจองสำเร็จ<br>• การยกเลิกการจอง<br><br>**SYSTEM:** ระบบตัดคิวอัตโนมัติ | **Frontend:** `booking.html`, `confirm.html`, `myticket.html`<br><br>**Backend:** `backend/routes/doctors.py`<br>`backend/routes/bookings.py`<br><br>**API:**<br>• `GET /api/doctors`, `GET /api/departments`<br>• `POST /api/bookings`<br>• `DELETE /api/bookings/{booking_id}` |
| **3** | **นายธีรัช มิฉะนั้น (คิม)** | **[ Home, Datetime ]**<br><br>• Main Dashboard: หน้าหลักแสดงเมนู<br>• Unified Admin Dashboard: หน้าจัดการรวมสำหรับเจ้าหน้าที่<br>• Select Schedule: หน้าเลือกวัน/เวลาจอง | **Frontend:** `home.html`, `admin.html`, `booking.html`<br><br>**Backend:** `backend/routes/doctors.py`<br><br>**API:**<br>• `GET /api/doctors`, `GET /api/departments`<br>• `POST /api/admin/login`<br>• `GET /api/doctors/{id}/slots` |
| **4** | **นายทีปกรณ์ แก่นกุล (ปอน)** | **[ Datetime, Details ]**<br><br>• Select Schedule: เลือกวัน/เวลาที่ต้องการจอง<br>• Booking Details: หน้าดูรายละเอียดการจอง<br>• Reschedule Details: เลือกวันและเวลาเพื่อเลื่อนการจอง | **Frontend:** `booking.html`, `details.html`, `reschedule.html`<br><br>**Backend:** `backend/routes/doctors.py`<br>`backend/routes/bookings.py`<br><br>**API:**<br>• `GET /api/admin/slots`<br>• `GET /api/bookings/{booking_id}`<br>• `PUT /api/bookings/{booking_id}` |
| **5** | **นายวุฒิภัทร วิริยเสนกุล (ชันเด)** | **[ History, Notification ]**<br><br>• Booking History: หน้าดูประวัติการจองทั้งหมด<br>• Notification Center: หน้าดูรายการแจ้งเตือน<br><br>**SYSTEM:** ระบบแจ้งเตือน | **Frontend:** `history.html`, `notification.html`<br><br>**Backend:** `backend/routes/bookings.py`<br>`backend/routes/notifications.py`<br><br>**API:**<br>• `GET /api/bookings`<br>• `GET /api/notifications` |
| **6** | **นายพงศกร กอคูณ (ฟิล์ม)** | **[ QR Code Scan, Cancel Booking ]**<br><br>• ดูรายการจองทั้งหมด (All Bookings)<br>• ตรวจสอบรายชื่อการจอง (Check Booking List) | **Frontend:** `admin.html`<br><br>**Backend:** `backend/routes/bookings.py`<br><br>**API:**<br>• `GET /api/bookings`<br>• `PUT /api/bookings/{booking_id}` |
| **7** | **นายรัชชานนท์ อรรถพันธ์ (แฟรงค์)** | **[ Admin ]**<br><br>• ตรวจสอบรายชื่อเข้าตรวจ<br>• ดูประวัติการจองทั้งหมด<br>• จัดการสล็อตเวลา (Manage Slots) | **Frontend:** `admin.html`<br><br>**Backend:** `backend/routes/bookings.py`<br><br>**API:**<br>• `POST /api/admin/slots`<br>• `PUT /api/admin/slots/{slot_id}`<br>• `DELETE /api/admin/slots/{slot_id}` |
| **8** | **นายอภิสิทธิ์ พรหมมา (ภีม)** | **[ Admin ]**<br><br>• Admin Home<br>• จัดการรายชื่อแพทย์ (Manage Doctors สร้าง/อัปเดต/ลบ) | **Frontend:** `admin.html`<br><br>**Backend:** `backend/routes/admin.py`, `backend/routes/doctors.py`<br><br>**API:**<br>• `GET /api/doctors`<br>• `POST /api/admin/doctors`<br>• `PUT /api/admin/doctors/{doctor_id}`<br>• `DELETE /api/admin/doctors/{doctor_id}` |
